# 首页接口文档

> ##主入口 http://magic-era.top/doctor

## 1. 推荐接口

- **接口路径**：`/recommend`  
- **请求方式**：GET  
- **接口说明**：获取推荐内容列表  
- **请求参数**：无  

**返回示例**：

```json
[
    {
        "id": 1,
        "title": "保持健康生活习惯",
        "content": "每天运动30分钟",
        "publish_time": "2025-12-01 10:00:00",
        "is_follow": 0
    },
    {
        "id": 2,
        "title": "预防皮肤干燥",
        "content": "多喝水，保持皮肤湿润",
        "publish_time": "2025-12-02 12:00:00",
        "is_follow": 1
    }
]
```

## 2. 护肤接口

- **接口路径**：`/skin`  
- **请求方式**：GET

```json
        "id": 1,
        "title": "冬季护肤小贴士",
```

```json
    }
]
```

## 5. 关注接口

- **接口路径**：`/follow`  
- **请求方式**：GET  
- **接口说明**：获取用户关注的内容，整合推荐、护肤、减脂、饮食里面所有用户已经关注的 
- **请求参数**：无  

**返回示例**：

```json
[
    {
        "id": 2,
        "title": "预防皮肤干燥",
        "content": "多喝水，保持皮肤湿润",
        "publish_time": "2025-12-02 12:00:00",
        "is_follow": 1,
        "source_type": "suggest"
    },
    {
        "id": 3,
        "title": "减脂餐计划",
        "content": "早餐燕麦，午餐鸡胸肉，晚餐蔬菜",
        "publish_time": "2025-12-01 08:30:00",
        "is_follow": 1,
        "source_type": "weight"
    }
]
```

**说明**：

`source_type` 标识数据来源：

`suggest` 推荐

`skin` 护肤

`weight` 减脂

`diet` 饮食

按 `publish_time` 降序排列。
